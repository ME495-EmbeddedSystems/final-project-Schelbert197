drawing.path_plan_execute module

Allow user to manipulate a robot.

Interface with the node and the robot to allow the user to manipulate the robot using the move group and action clients.

Services:

none

Clients:

  • compute_fk (GetPositionFK) - Calls the FK service for forward kinematics.

  • compute_ik (GetPositionIK) - Calls the IK service for inverse kinematics.

  • compute_cartesian_path (GetCartesianPath) - Calls a service to compute a cartesian path.

Actions:

  • move_action (MoveGroup) - The action for asking MoveIT to plan non- cartesian paths.

  • panda_gripper/homing (Homing) - The action for homing the panda gripper.

  • panda_gripper/grasp (Grasp) - The action for grasping with the panda gripper.

Publishers:

  • /collision object (CollisionObject) - The collision box representing the table

Subscribers:

  • /joint_states (JointState) - The current joint state of the robot.
