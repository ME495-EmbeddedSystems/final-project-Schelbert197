drawing.hangman module

The Hangman Node.

Plays the hangman game based on the OCR user input. Interfaces with the brain node and the OCR node to evaulate data.

PUBLISHERS:

  • /writer (LetterMsg) - The data sent to brain for a given play.

SUBSCRIBERS:

  • /user_input (String) - The guess sent from OCR for given play.
